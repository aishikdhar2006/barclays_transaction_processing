Thank you for downloading BankingPlatform tools for OSX!!

This is binary, ready to use. You just need to give execution permissions to it with chmod:

chmod +x banking_tools

If you see the error "banking_tools is damaged and can’t be opened", try to clear the extended attributes:

xattr -c banking_tools

Check your available options with ./banking_tools --help

For examples, see https://github.com/banking/banking_tools/blob/master/README.md

If you need help, contact us at support@banking.com.
